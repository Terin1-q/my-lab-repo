# Основные команды Git

## git init
**Назначение**: Инициализирует новый Git репозиторий в текущей директории.

**Пример**:
```bash
mkdir my-project
cd my-project
git init
```

## git clone
**Назначение**: Клонирует удаленный репозиторий на локальную машину.

**Пример**:
```bash
git clone https://github.com/user/repository.git
```

## git add
**Назначение**: Добавляет изменения в индекс (staging area) для следующего коммита.

**Примеры**:
```bash
# Добавить конкретный файл
git add filename.txt

# Добавить все изменения
git add .

# Добавить все файлы с расширением .js
git add *.js
```

## git commit
**Назначение**: Создает коммит с проиндексированными изменениями.

**Пример**:
```bash
git commit -m "Добавлена новая функциональность"
```

## git push
**Назначение**: Отправляет коммиты в удаленный репозиторий.

**Пример**:
```bash
# Отправить в ветку main
git push origin main

# Отправить в ветку с определенным именем
git push origin branch-name
```

## git pull
**Назначение**: Забирает изменения из удаленного репозитория и объединяет с локальным.

**Пример**:
```bash
git pull origin main
```
